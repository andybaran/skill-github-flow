Add a dark-mode toggle to the settings page so users can switch between light and dark themes.

## Acceptance criteria

- Toggling it switches the page between light and dark themes.
- The chosen theme persists across reloads (localStorage).
- On first visit, the theme respects the OS `prefers-color-scheme` preference.
