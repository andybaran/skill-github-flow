To list your local branches:

```
git branch
```

The current branch is marked with a `*`.

Handy variants:
- `git branch -v` — local branches with the latest commit on each
- `git branch -a` — local **and** remote-tracking branches
- `git branch -r` — remote-tracking branches only
- `git branch --merged` / `--no-merged` — branches already merged (or not) into your current branch
